
1. Eu escolhi usar signal(false) no lugar de uma propriedade comum porque o estaCarregando é um estado de UI, e Signals foram feitos exatamente para isso. A grande vantagem é que o Angular sabe exatamente onde esse valor é lido no template e atualiza apenas aquele trecho do DOM, sem verificar o componente inteiro. É mais performático e mais intencional, fica claro que aquilo é estado, não um dado derivado de um evento.
2. Para atualizar o Signal dentro do fluxo RxJS, usei .set() dentro do tap e do finalize. O tap 'espia' o fluxo e liga o loading antes da requisição disparar, sem interferir nos dados. Já o finalize garante que o loading seja desligado sempre, seja a requisição um sucesso ou um erro. É a nossa apólice de seguro: o spinner jamais ficará travado na tela.
3. O passo final foi usar o toSignal() para transformar o Observable de resultados em um Signal. Isso elimina o async pipe do template e nos dá um código mais limpo e homogêneo: o template passa a ler dois Signals, sem precisar entender o que é Observable ou o que é dado comum. O Angular cuida da inscrição internamente e cancela tudo automaticamente quando o componente é destruído. É a ponte perfeita entre o mundo do RxJS e o mundo dos Signals.

#### Componente
```typescript
import { Component, OnInit, signal, inject } from '@angular/core';
import { FormControl } from '@angular/forms';
import { Observable, debounceTime, distinctUntilChanged, switchMap, tap, finalize, of } from 'rxjs';

@Component({ ... })
export class BuscaPessoasComponent implements OnInit {
 estaCarregando = signal(false);
  searchControl = new FormControl('');
 resultados$!: Observable<any[]>;
 private pessoaService = inject(PessoaService);
 ngOnInit(): void {
   this.resultados$ = this.searchControl.valueChanges.pipe(
     debounceTime(500),
     distinctUntilChanged(),
     tap(() => this.estaCarregando.set(true)),
     switchMap(termo => {
       if (!termo?.trim()) {
          this.estaCarregando.set(false);
          return of([]);
       }
       return this.pessoaService.buscar(termo).pipe(
         finalize(() => this.estaCarregando.set(false))
       );
     })
   );
 }
}

```

#### Template
```html
<div *ngIf="estaCarregando()">Carregando resultados...</div>
<ul *ngIf="!estaCarregando()">
 <li *ngFor="let pessoa of resultados$ | async">
   {{ pessoa.nome }}
 </li>
</ul>
```

#### Serviço
```typescript
import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable, delay } from 'rxjs';
@Injectable({ providedIn: 'root' })
export class PessoaService {
 constructor(private http: HttpClient) {}
 buscar(termo: string): Observable<any[]> {
   return this.http.get<any[]>(`https://api.exemplo.com/pessoas?q=${termo}`);
 }
}
```
