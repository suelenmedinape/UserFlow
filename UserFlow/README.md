# UserFlow — Frontend Project

Este diretório contém o código-fonte da aplicação Angular.

Para documentação completa, instruções de execução (Docker/Local) e detalhes da arquitetura, consulte o **[README principal na raiz do projeto](../README.md)**.

## Atalhos rápidos:

- **Instalar:** `npm install`
- **Rodar:** `npm start`
- **Testar:** `npm test`
- **Build:** `npm run build`
